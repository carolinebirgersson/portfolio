My thoughts about working with code

Why did you start coding?

During my work as a senior ux writer I often worked in teams with designers and developers. My role was to tell the developer how to structure the content on a website. While doing that over and over again, I started to have a wish of doing it myself, not always be in the hands of a developer trying to explain how I think.

What is the benefits of your background in UX Writing?

The benefit is that "content first" and the user is the core of my work. As a ux writer you always have the user in mind when you plan the site. You gather insights from analizing user data and export those insights into actions that responds to what the user needs. You work with SEO to make sure the user finds the content. And you write microcopy that is user friendly and navigates the user forward (hopefully towards conversion).

What are you aiming for in every project?

To create a website that is user friendly and accessible for everyone.
